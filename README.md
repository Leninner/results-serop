# Senirop Challenges

## Description

This repository contains the solutions to the challenges proposed

## How to use

Requirements:

- Node.js >= 20
- pnpm

Install the dependencies:

```bash
pnpm install
```

Run the tests:

```bash
# Run all tests
pnpm test 

# Run the coverage
pnpm coverage
```

Run the solutions:

```bash
npm run start
```
